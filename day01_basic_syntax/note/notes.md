1. Python là gì?
Python là một ngôn ngữ lập trình:

Dễ học

Cú pháp đơn giản, giống tiếng Anh

Rất mạnh, dùng để tạo website, AI, phân tích dữ liệu, game…

2. Cách cài Python
Bạn làm lần lượt:

Bước 1: Tải Python

Vào trang: https://www.python.org/downloads

Chọn Download Python for Windows.

Bước 2: Khi cài nhớ tick:

✔️ Add Python to PATH

Nếu thiếu bước này thì bạn sẽ không chạy lệnh được.

3. Mở môi trường chạy Python

Có 2 cách dễ:

Cách 1: Dùng Terminal

Mở CMD

Gõ thử: python --version
